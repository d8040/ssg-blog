      **1\. 반올림 ( 자르기 포함 )** **함수 : ROUND ( numeric\_expression, length\[, function\])** **1) 반올림** ``` ---------------------------------------------- -- 반올림 ---------------------------------------------- SELECT ROUND(12.35393,2) --> 결과값 : 12.35000 SELECT ROUND(12.35593,2) --> 결과값 : 12.36000 SELECT ROUND(12.35393,-1) --> 결과값 : 10.00000 SELECT ROUND(15.35393,-1) --> 결과값 : 20.00000 SELECT ROUND(12.35393,0) --> 결과값 : 12.00000 ``` **2) 자르기** ``` ---------------------------------------------- -- 자르기 ---------------------------------------------- Select Round(98.65, 1, 1) --> 결과값 : 98.60 Select Round(98.65, 0, 1) --> 결과값 : 98.00 Select Round(98.65, -1, 1) --> 결과값 : 90.00 ``` **2\. 올림** **함수 : CEILING ( numeric\_expression )** **<예>** ``` ---------------------------------------------- -- 올림 ---------------------------------------------- SELECT CEILING(12.35393) --> 결과값 : 13 SELECT CEILING(-12.35393) --> 결과값 : -12 SELECT CEILING(0.00) --> 결과값 : 0 ``` **3\. 내림** **함수 : FLOOR ( numeric\_expression )** **<예>** ``` ---------------------------------------------- -- 내림 ---------------------------------------------- SELECT FLOOR(12.35393) --> 결과값 : 12 SELECT FLOOR(-12.35393) --> 결과값 : -13 SELECT FLOOR(0.00) --> 결과값 : 0 ```
